# RoutePick 실행 방법

## 1단계: 의존성 설치

터미널에서 다음 명령어를 실행하세요:

```bash
pip install -r requirements.txt
```

## 2단계: 환경 변수 설정

`.env` 파일을 생성하고 다음 내용을 입력하세요:

```env
TAVILY_API_KEY=your_tavily_api_key
GOOGLE_MAPS_API_KEY=your_google_maps_api_key
OPENAI_API_KEY=your_openai_api_key
LLM_MODEL=gpt-4
```

> **참고**: `.env.example` 파일을 참고하여 `.env` 파일을 생성할 수 있습니다.

### API 키 발급 방법:
- **Tavily API**: https://tavily.com 에서 API 키 발급
- **Google Maps API**: https://console.cloud.google.com/ 에서 Google Maps API 키 발급
- **OpenAI API**: https://platform.openai.com/api-keys 에서 API 키 발급

## 3단계: 예제 실행

```bash
python example_usage.py
```

## 또는 직접 Python 코드 실행

```python
import asyncio
from agents import SearchAgent, PlanningAgent, RoutingAgent
from config.config import Config

async def main():
    config = Config.get_agent_config()
    
    # Search Agent 실행
    search_agent = SearchAgent(config=config)
    result = await search_agent.execute({
        "theme": "비 오는 날 서울 실내 데이트",
        "location": "서울",
        "max_results": 20,
        "min_rating": 4.0
    })
    print(result)

if __name__ == "__main__":
    asyncio.run(main())
```

## 문제 해결

### 모듈을 찾을 수 없는 오류
- 가상환경을 사용하는 경우 가상환경이 활성화되어 있는지 확인하세요
- `pip install -r requirements.txt`로 모든 의존성이 설치되었는지 확인하세요

### API 키 오류
- `.env` 파일이 프로젝트 루트 디렉토리에 있는지 확인하세요
- API 키가 올바르게 입력되었는지 확인하세요
- 각 API 서비스에서 API 키가 활성화되어 있는지 확인하세요
